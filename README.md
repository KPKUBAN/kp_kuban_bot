# kp_kuban_bot

**Telegram-бот для КП-Кубань**  
Генерирует стилизованные посты из [kuban.kp.ru](https://kuban.kp.ru) на бесплатной модели Flan-T5.

---

## Возможности

- **Авто-анонс**: проверяет RSS kuban.kp.ru и публикует новые статьи в канал  
- **Стилизация**: переписывает заголовок, лид и текст в стиле КП-Кубань (с эмодзи, короткими абзацами)  
- **Инлайн-режим**: `@kp_kuban_bot <ссылка>` — сразу выдаёт готовый превью-пост  
- **Дайджест**: `/digest` — топ-5 ссылок за последнюю неделю  
- **Аналитика**: логирует публикации и каждую неделю шлёт отчёт в личный чат
